# Smart Unit Converter

A modern Python GUI Unit Converter built using Tkinter.

## Features
- Clean dark theme GUI
- Convert multiple unit types
- Length conversion
- Weight conversion
- Temperature conversion
- Time conversion
- Speed conversion
- Simple and beginner-friendly code
- Error handling for invalid input

## Supported Categories

### Length
- Meter
- Kilometer
- Centimeter
- Millimeter
- Mile
- Yard
- Foot
- Inch

### Weight
- Kilogram
- Gram
- Milligram
- Pound
- Ounce

### Temperature
- Celsius
- Fahrenheit
- Kelvin

### Time
- Second
- Minute
- Hour
- Day

### Speed
- m/s
- km/h
- mph
